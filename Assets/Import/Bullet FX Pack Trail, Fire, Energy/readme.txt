# Bullet FX Pack: Trail, Fire & Energy

## Description
This pack contains a single effect prefab with three visual modes:
- Trail
- Fire
- Energy

Compatible with the Built-in Render Pipeline.

## Usage
The effect type is selected from a dropdown list on the bullet prefab,  
in the `Bullet` component.

## Demo Scene
Open `Scenes/DemoScene.unity` to see the effect in action.

## Notes
Different visual modes are intended for different projectile speeds.  
Experiment with bullet speed for correct visual appearance.

## Location
The prefab is located in `Assets/Bullet FX Pack Trail, Fire, Energy/Prefabs/`.

Tested with Unity 6 (6000.0.39f1).
